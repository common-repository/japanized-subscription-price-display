Modifies default WooCommerce Subscriptions prices to look more natural in Japanese.
WooCommerce Subscriptionsのデフォルト価格表示を、より自然な日本語のフォーマットに変換します。
 
== Description ==

WooCommerce Subscriptionsのデフォルト価格表示を、より自然な日本語のフォーマットに変換します。

例）
*デフォルト*

>最低価格: ¥1,980 税別 / 月

*変換後*

>月々 ¥1,980より 税別


== Installation ==

1. Upload `japanized-subscription-price-display` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Adjust JA translations of WooCommerce and WooCommerce Subscriptions as desired

== Frequently Asked Questions ==

= Can I use it for non-Japanese language sites? =

Yes, you can. But the price might appear a little odd for you. Please see the Description for more info.

= WooCommerceの日本語訳を変更するにはどうすればよいですか？ =

「Loco Translate」プラグインを試してみてください。
